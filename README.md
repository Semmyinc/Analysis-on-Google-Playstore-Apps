Google Play Store Apps Analysis with Pandas

A data analysis project that examines Google Play Store apps data using Python and Pandas to uncover insights about app categories, ratings, installs, pricing models, and market trends.

Overview

This project explores the Google Play Store dataset to identify patterns in app performance and user engagement. Through data cleaning, exploratory analysis, and visualization, the project highlights factors that influence app popularity and user ratings.

Objectives

Clean and preprocess the Play Store dataset

Analyze app categories and their distribution

Examine relationships between ratings, reviews, and installs

Identify top-performing app categories

Compare free vs paid app trends

Tech Stack

Python

Pandas

NumPy

Matplotlib / Seaborn

Jupyter Notebook

Key Insights Explored

Most popular app categories by installs

Rating distribution across categories

Impact of reviews on app ratings

Free vs paid app performance

Apps with the highest engagement
